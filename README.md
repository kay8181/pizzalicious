Pizza Webshop Order Intake System, A backend-focused order intake plaform for managing order operations. This README outlines the systom's purpose.
Overview: the Pizza Webshop Order Intake System is designed for pizza shop owners who need a centralized dashboard to process orders and file receipts. The system supports online transactions. 
Features
Responsive UI – Takes user input

Customizable Orders – Choose size, crust, toppings, and quantity

Checkout Flow – Order total and order confirmation

Receipt Filer - Files receipt orders
